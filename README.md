# LA STATION DU CODE

## 🚀 Hub d'Innovation Positive | Renaissance Numérique

"LA STATION DU CODE" est une plateforme immersive conçue comme un Hub d'Innovation pour une validation RNCP Niveau 7. Elle transmute l'exigence de la logistique physique vers la maîtrise architecturale logicielle.

### 🌟 Expérience Immersive
- **Atrium Central** : Une interface 3D WebGL (React Three Fiber) symbolisant le cœur du réacteur.
- **Blueprint Mode** : Transformation interactive de la station en schéma technique filaire.
- **Piliers d'Expérience** : Pages dédiées à la Cuisine (Dev), l'Observatoire (Data) et le Cœur (Résilience).

### 🛠️ Mode Expert & Architecture
- **Expert Terminal** : Appuyez sur **`T`** pour activer l'overlay technique documentant les compétences et l'architecture du système.
- **Technical Blueprint** : Page `[Blueprint]` détaillant la stack technique et les flux de données.
- **Backend Service** : Intégration en temps réel avec InsForge (PostgreSQL) pour la logistique des données.

### 💻 Installation & Lancement

```bash
# Installation des dépendances
npm install

# Lancement du serveur de développement
npm run dev

# Construction pour la production
npm run build

# Lancement du serveur de production
npm run start
```
